# Powershell-Project
Simple Powershell projects Which gives report on installed software in a system
Purpose:
	This PowerShell script audits installed software on a Windows workstation and enables users to filter results by installation date. It helps system administrators verify installation dates, troubleshoot recent changes, and generate compliance reports. The script offers interactive prompts and an option to export results to a text file with user input date a Result folder, supporting both quick reviews and formal documentation.
